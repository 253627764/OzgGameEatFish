ozg game—大鱼吃小鱼


================

本游戏主要用于技术的研究和积累，游戏中所用的图片和声音文件均来自互联网


使用cocos2d-x 3.1.1，win32、ios、android（真机可运行）编译通过。android编译命令，cocos compile -p android -j 4

目前只做了英文版

本游戏的cocos2d-iphone版本：https://github.com/ouzhigang/ozggame_eat_fish
